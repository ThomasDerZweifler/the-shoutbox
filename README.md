# The Shoutbox

It's a simple AngularJS application.

# Installation

1. Install [node.js and npm](http://nodejs.org/download/ "Download node.js")
2. $ git clone git@github.com:nosch/ng-shoutbox "shoutbox"
3. $ npm install

# Launch Application

1. $ node server.js
2. Open http://localhost:3000 in your browser.
